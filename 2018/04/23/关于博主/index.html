关于博主的一些信息

SimonKing[恒子]. 就读于福州的某不知名的技校，于 2017 年入学，主修计算机。

最初是因为看了这篇文章the_start，受到启发，才决定搭起博客。对于博客主要的内容，就目前来说，可能是想写什么就写什么吧，所以更新博客什么的，可能很勤，也可能很懒。

接触过一段时间的 Linux。刚开始用 Xubuntu，后来用了一年多的 Fedora。中间折腾过 Arch，不过时间很短。然而现在用 Mac。

略宅。每天都看动漫，有时也会去了解ACG的最新消息，但其实对ACG并不算很了解。

接触编程不算久，仔细算算，也才在大学是开始学习编程。与高中就开始编程的人相比，我还需要学习很多内容。

对 Python (Web 后端方向) 与 JavaScript (Web 前端方向) 接触相对多一点，不过还在努力学习中。

去年(2016)年，花了大半年的时间，考了 3 次 TOEFL 和 2 次 GRE。现在正在申请美帝的 CS master，不过目前结果未知。
